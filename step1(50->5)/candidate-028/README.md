# The One-Arm AI Slot Machine Experiment

A satirical slot machine application built with vanilla web technologies (HTML, CSS, JS).

## Features
- **5-Reel "Mega-Model"**: High complexity matching for "Synthetic Gains".
- **Retro-Arcade Aesthetic**: Neon colors, pixel fonts, and CSS animations.
- **Compute Token Economy**: Spend tokens to "train" the model and win more to keep the experiment running.
- **AI Satire**: Messages that make fun of AI hype, hallucinations, and "stochastic parrots".

## Symbols
- 🧠 **AGI**: The ultimate (but rare) jackpot.
- ⚡ **GPU**: High-performance compute units.
- ⛓️ **Blockchain**: A pivot that might not pay off.
- 👻 **Hallucination**: High frequency, low value.
- 🚫 **Censorship**: Safety layers that limit outcomes.
- 💰 **VC Funding**: Necessary for more training.
- 🤖 **Chatbot**: The most common result.

## How to Run
Simply open `index.html` in any modern web browser.
- Click **TRAIN (SPIN)** to spend 10 Compute Tokens.
- Click **INSERT COIN** if you run out of tokens (VC funding to the rescue!).
