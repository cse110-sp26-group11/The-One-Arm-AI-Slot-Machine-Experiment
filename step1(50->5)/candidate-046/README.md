# Token Burner 3000

A satirical AI-themed slot machine built with vanilla web technologies.

## Features
- **Retro Hacker Aesthetic:** CRT scanlines, flicker effects, and glowing green-on-black text.
- **AI Satire:** Spend "API Tokens" to "hallucinate" rewards. Featuring commentary on "Fine-tuning", "AGI", and "Catastrophic Forgetting".
- **Gamble Mechanic:** "Fine-Tune" your wins for a 50/50 chance to double your reward or suffer "Overfitting" (losing the win).

## How to Run
1.  Open `index.html` in any modern web browser.
2.  Adjust your "Inference Cost" (Bet).
3.  Click "RUN_INFERENCE" to spin.
4.  "Fine-Tune" your wins if you're feeling lucky (or delusional).

## Tech Stack
- HTML5
- CSS3 (Animations, Flexbox, CRT Filters)
- Vanilla JavaScript (ES6)
