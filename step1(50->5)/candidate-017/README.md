# AI Slot Machine: The Compute Console

A satirical web-based slot machine themed around the AI "token economy."

## Features
- **Retro-Terminal Aesthetic**: Dark mode with glowing green accents and scanline effects.
- **Satirical Log**: A terminal-style output that pokes fun at AI buzzwords, hallucinations, and ethics.
- **Token Economy**: Spend "Compute Tokens" to "Generate" results.
- **Reel Symbols**:
    - 🤖 (AI Model) - Jackpot
    - ⚡ (Compute)
    - 🍄 (Hallucination)
    - 💰 (Profit)
    - 🔥 (GPU Melt)
    - 📉 (Model Collapse)
    - ✨ (Sparkles/Magic)

## How to Play
1. Open `index.html` in any modern web browser.
2. Click **GENERATE** to spend 10 tokens and spin the reels.
3. Try to achieve "Singularity" (3x 🤖) or other winning combinations.
4. Watch the console log for system updates and satirical messages.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Grid, Transitions, Animations)
- Vanilla JavaScript (ES6+, Promises, Async/Await)
